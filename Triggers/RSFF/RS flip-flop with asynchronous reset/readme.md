# RS触发器

实现了上升沿触发的RS触发器的功能。**上升沿**来临时RS触发器实现的功能如下表所示：

| R    | S    | 功能         | Q    |
| ---- | ---- | ------------ | ---- |
| 0    | 0    | 保持         | Q    |
| 0    | 1    | 置1          | 1    |
| 1    | 0    | 置0          | 0    |
| 1    | 1    | 禁止此种输入 | \    |

对于RS触发器，输入"R"即代表同步的复位功能。而在“RSFF_2.v”中，另增加一个异步复位端rst_asyn, 当复位信号rst_asyn由0变为1时，输出Q立刻置0.
