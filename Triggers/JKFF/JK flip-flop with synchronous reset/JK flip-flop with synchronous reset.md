# JK触发器（同步复位）

实现了上升沿触发的JK触发器的功能。**上升沿**来临时JK触发器实现的功能如下表所示：

| J    | K    | 功能 | Q    |
| ---- | ---- | ---- | ---- |
| 0    | 0    | 保持 | Q    |
| 0    | 1    | 置0  | 0    |
| 1    | 0    | 置1  | 1    |
| 1    | 1    | 翻转 | ~Q   |

“JKFF_1.v”中，实现的是带有同步复位端的JK触发器（复位端1有效），再时钟信号clk上升沿时，若采到rst=1，则输出Q置0.
